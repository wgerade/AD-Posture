## Use Cases

- Review privileged access paths before an audit, merger, incident, or remediation project.
- Identify risky nested access into Domain Admins, Enterprise Admins, Administrators, DnsAdmins, Backup Operators, Account Operators, Server Operators, Print Operators, Key Admins, Enterprise Key Admins, Exchange groups, and other sensitive groups.
- Prioritize Tier 0 identity exposure and reduce blast radius in an on-premises Active Directory environment.
- Produce local CSV, JSON, and static dashboard evidence for identity security reviews.
- Support Microsoft Defender for Identity, SIEM, EDR, GRC, and compliance workflows with local posture evidence.
- Review ADCS, Kerberos/Auth, GPO, ACL, Trust, and DNS posture without requiring an agent or continuous collector.
- Generate review-ready remediation context and WhatIf-safe remediation scripts for approved change workflows.
